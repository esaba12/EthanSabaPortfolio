<objective>
Wire the Plan 01-01 foundation into the shared page shell: mount the persistent blueprint
grid and the Lenis smooth-scroll provider once in the root layout so they apply sitewide,
swap the unused Space Grotesk display-font import for Chakra Petch, and restyle the shared
navbar/footer off the old glassmorphism-glow treatment onto the blueprint system's hairline
aesthetic.

Purpose: The UI-SPEC requires the grid and smooth scroll to be "truly persistent and
identical across all pages," which only a single root-layout mount can guarantee — mounting
them per-page would violate that requirement and would need to be redone in every future
phase. This plan also satisfies Roadmap Success Criterion #3 ("shared shell reflects the new
system and is reused, not page-specific") ahead of the Home page rebuild in Plan 01-03, so
the final checkpoint in that plan can show the complete, correctly-shelled page.
Output: Updated root layout with the new font/grid/scroll wiring, restyled navbar and footer.
</objective>
